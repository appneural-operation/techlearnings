# 🚀 Tech Learnings

Welcome! 👋

This repository is my public learning journal where I document technical concepts as I learn them.

I created this repository with a simple idea: whenever I learn something new in technology, I'll document my understanding and upload it here. There is no fixed roadmap, no predefined syllabus, and no specific technology stack. This repository will grow naturally with my learning journey.

The goal is to learn in public, build consistency, and create a knowledge base that anyone can explore and benefit from.

## 📖 About This Repository

* Every document is based on what I'm currently learning.
* Topics may vary depending on what I explore next.
* Notes may be updated or improved as my understanding evolves.
* The focus is on understanding concepts and sharing knowledge.

## 🤝 Contribute

If you find something that can be improved, notice an error, or have a better explanation, feel free to open an Issue or submit a Pull Request.

Learning is a continuous process, and every contribution is appreciated.
